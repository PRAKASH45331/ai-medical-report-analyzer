# 🌐 Get Your AI Medical Report Analyzer Website Link

## 📋 Step-by-Step Instructions

### Step 1: Install Required Tools (5 minutes)

**Install Git:**
1. Go to: https://git-scm.com
2. Download and install Git
3. Restart your computer

**Install Heroku CLI:**
1. Go to: https://devcenter.heroku.com/articles/heroku-cli
2. Download and install Heroku CLI
3. Open Command Prompt and type: `heroku login`

### Step 2: Create Heroku Account (2 minutes)

1. Go to: https://signup.heroku.com
2. Create free account
3. Verify your email

### Step 3: Get OpenAI API Key (2 minutes)

1. Go to: https://platform.openai.com
2. Create account or login
3. Go to API Keys section
4. Create new API key
5. Copy the key (starts with "sk-")

### Step 4: Deploy Your Website (3 minutes)

1. **Double-click `quick-deploy.bat`**
2. **Enter your OpenAI API key** when prompted
3. **Wait for deployment** (2-3 minutes)
4. **Get your live website link!**
